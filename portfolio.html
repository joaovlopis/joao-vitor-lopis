Projeto: APP de Vendas

No curso de Lógica de Programação e Algoritmos, desenvolvi um sistema simples que emula o funcionamento de um aplicativo de vendas para uma empresa. Este projeto teve como objetivo aplicar os conceitos fundamentais de lógica, fluxo condicional e operações matemáticas.

Descrição do Sistema

O sistema, denominado "Loja do João Vitor Lopis Neves", realiza as seguintes operações:

  • Entrada de Dados: Solicita o preço unitário do produto e a quantidade desejada.
  • Cálculo do Valor Total: Multiplica o valor unitário pela quantidade.
  • Aplicação de Descontos: Aplica um desconto com base no valor total da compra, conforme as seguintes faixas:
    • Sem desconto para compras abaixo de R$ 2.500.
    • Desconto de 4% para compras entre R$ 2.500 e R$ 6.000.
    • Desconto de 7% para compras entre R$ 6.001 e R$ 10.000.
    • Desconto de 11% para compras acima de R$ 10.000.
  • Exibição de Resultados: Exibe o valor total da compra, o desconto aplicado e o valor final com o desconto.

Código do Sistema

O código do projeto é apresentado abaixo, com destaque e explicações:


# Mensagem de boas-vindas
print('Bem-Vindo à Loja do João Vitor Lopis Neves 4840374!')

# Entrada do valor unitário e quantidade do produto
valor_unitario = float(input('Entre com o valor unitário do produto: R$'))
qtd = int(input('Entre com a quantidade do produto: '))

# Cálculo do valor total sem desconto
valor_total = valor_unitario * qtd

# Implementação do desconto conforme as condições
if valor_total < 2500:
    desconto = 0
elif valor_total <= 6000:
    desconto = 4
elif valor_total <= 10000:
    desconto = 7
else:
    desconto = 11

# Cálculo do valor total com desconto
valor_com_desconto = valor_total - (valor_total * desconto / 100)

# Mensagens de saída com os resultados
print('O valor sem desconto foi: R${:.2f}'.format(valor_total))
print('O valor com desconto foi: R${:.2f} (desconto {:.0f}%)'.format(valor_com_desconto, desconto))

Resultados Obtidos

Este projeto permite que o usuário realize compras e veja o impacto dos descontos, tornando-se uma aplicação simples, mas funcional para vendas online. Além disso, a implementação ajudou a entender melhor a lógica de programação e a estruturação de processos lógicos em Python.
